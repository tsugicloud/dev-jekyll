Accessing this Server

For wide open dev servers the following are the defaults for access.

  • Tsugi admin: the mascot of the Sakai project - all lower case

  • Database account: The last part of your url for example if the url was

    • https://dev.tsugicloud.org/d/zap, the database user would be 'zap'

  • Database password: db_ followed by the admin password like db_happy

If these do not work, contact the creator of this site.
